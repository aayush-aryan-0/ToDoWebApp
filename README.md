### Overview

A multi-user task management web application built with Python and Flask.
Users can create, update, and delete tasks while receiving reminder notifications through Telegram integration.
### Features
- User authentication system for multiple users

- Create, update, delete, and manage tasks

- Task completion tracking

- Persistent task storage using SQLite

- Telegram API integration for reminder notifications

- Simple web interface built with HTML5 and CSS

### Tech Stack

- Backend: Python, Flask

- Frontend: HTML5, CSS

- Database: SQLite

- Notifications: Telegram Bot API
